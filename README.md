# Worldview Match v1

シックで高級感のあるBtoB向けクリエイターマッチングシステムの本番制作ベースです。

## 起動方法

`index.html` をブラウザで開くだけで動きます。

## 現在できること

- 企業向けホーム画面
- 案件登録
- 案件一覧
- 世界観タグによるクリエイター検索
- 案件提案
- 提案状況の表示
- クリエイター一覧表示
- ブラウザ内保存

## 次に本番化する時に追加するもの

- 企業ログイン
- クリエイターログイン
- 運営ログイン
- データベース保存
- 画像アップロード
- 面談日程調整
- 契約ステータス管理
- 管理者審査画面

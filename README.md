# EIRAUM MAP Demo Ver.1.0

EIRAUM MAPのデモです。  
これはユーザー向け画面ではなく、開発・設計用の「MAPモード」です。

## 構成

- `index.html`
- `style.css`
- `script.js`

## GitHub Pagesで公開する手順

1. GitHubで新しいリポジトリを作成
2. このフォルダ内のファイルをアップロード
3. Repository Settings → Pages
4. Branchを `main` / root に設定
5. 表示されたURLを開く

## EIRAUM開発ルール

- 世界創造モード：ユーザーが見る体験を作る
- MAPモード：EIRAUMの脳・設計図を作る
- MAPに存在しない機能は作らない
- すべての機能は思想と接続する

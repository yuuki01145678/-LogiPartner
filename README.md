# EIRAUM v2.1 Dev

## 変更内容
- ブランド表記を EIRAUM に変更
- 起動時の大きなロゴを EIRAUM に変更
- ログイン画面からデモ用パスワード表示を削除
- パスワード管理方針を明記
- パスワード再設定はオーナーのみの画面を追加

## パスワード運用方針
- クライアント企業：契約時に運営から発行
- 表現者：仮登録後、本登録時に発行
- 運営側：会社内でのみ共有
- パスワード再設定：オーナーのみ

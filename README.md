# LogiPartner Ver.1.0 Dev

管理者を楽にするための、CPT優先の滞留危険度チェックツールです。

## 現在できること

- カメラでバーコードスキャン
- テストスキャン
- CPTまでの残り時間を表示
- 滞留時間を表示
- 安全 / 注意 / 危険 の色分け
- 最終確認者と確認時刻を記録
- 管理者一覧で危険荷物から順番に表示
- ブラウザ内にデータ保存

## 公開方法

GitHub Pagesで公開する場合は、このフォルダ内のファイルをリポジトリにアップロードしてください。

- index.html
- style.css
- app.js

その後、GitHub Pagesを有効化するとスマホでカメラテストできます。

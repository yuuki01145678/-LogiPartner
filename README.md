## 権限方針

- オーナー：最高権限。全データ閲覧、履歴削除、サブオーナー管理が可能。
- サブオーナー：審査、面談調整、チャット対応が可能。履歴削除や権限管理は不可。
- 企業：案件登録、表現者検索、提案状況、案件チャット。
- 表現者：プロフィール登録、提案確認、案件チャット、マイ金庫。
